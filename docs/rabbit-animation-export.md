# 兔子动画导出规范

目标：把 `比卡丘动画导出/tuzi.psd` 导出为运行时可直接引用的 7 个 GIF，并放到 `兔子动画导出/`。

## 目录与文件名（固定）

请确保输出文件名与下面完全一致：

- `兔子动画导出/呼吸演示.gif`
- `兔子动画导出/等待演示.gif`
- `兔子动画导出/观察演示.gif`
- `兔子动画导出/互动演示.gif`
- `兔子动画导出/休息演示.gif`
- `兔子动画导出/吃演示.gif`
- `兔子动画导出/摸演示.gif`

## 导出建议

- 尺寸：与当前运行中的宠物显示比例一致（建议等比导出，避免拉伸）。
- 背景：透明背景。
- 循环：开启循环播放。
- 帧率：12-16 fps（动作快的可略高，休息/呼吸可略低）。
- 时长：保持各动作可辨识，避免过短造成闪烁。

## 验收方式

在项目根目录执行：

`npm run verify:rabbit-assets`

通过标准：

- 命令输出 `Rabbit asset check passed. All required files exist.`
- 宠物小屋内触发动作时，7 个状态均能正常显示，无透明边抖动、跳帧或拉伸异常。
